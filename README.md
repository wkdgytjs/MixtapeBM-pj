## ➰기획 배경 및 해결하고자 하는 문제
<table>
  <tr>
    <td>
여러 상품(음악 트랙) 목록이 한 눈에 들어오는 장점이 있고, 타 쇼핑몰과 달리 항목이 많지 않고 상품 이미지가 난잡하지 않아 차분한 느낌이 들었습니다.
또, 기존의 Mixtape Seoul에는 없는 아티스트들이 서로 소통을 할 수 있는 커뮤니티 게시판을 추가하여 구현해보았습니다.
    </td>
  </tr>
</table>
